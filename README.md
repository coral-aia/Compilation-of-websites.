# 灵感网站建设域

## 如何部署
1. 新建 GitHub 仓库（公开），将本文件夹所有内容上传。
2. 修改 `admin/config.yml` 中的 `repo` 为你的仓库地址。
3. 仓库 Settings → Pages → 分支选 `main`，根目录 `/` 保存。
4. 访问 `https://你的用户名.github.io/仓库名/admin`，用 GitHub 登录授权。
5. 开始添加/编辑网站，上传截图即可。

## 日常使用
- 前台：`https://你的用户名.github.io/仓库名`
- 后台：`https://你的用户名.github.io/仓库名/admin`

## 说明
- 截图建议宽度 ≥ 600px，比例 16:9 效果最佳。
- 分类在添加网站时自由填写，前端会自动生成筛选按钮。
